## live_push

### 项目内容
通过 [腾讯云函数](https://cloud.tencent.com/product/scf) 的函数部署实时监控多个直播间的状态，并使用 [leancloud](https://cloud.tencent.com/product/scf) 数据存储功能，存储相关直播数据,当检测某直播开始时候，通过 [push+](http://pushplus.hxtrip.com/) 进行信息传送



### 主要Python库
leancloud
requests
